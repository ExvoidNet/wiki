Fondé en Mai 2021 par Mathias Pepin-Leduc et Mathis Boulanger, Exvoid Network est une communauté québécoise ouverte et accueillante regroupant les amateurs de jeux multijoueur, tous nous serveur(s) de jeu sont hébergé localement sur notre propre équipement dédier.

Minecraft est présentement notre premier jeu avec plus à venir dans le futur!

## Void

Void est notre machine d'hébergement, un DELL PowerEdge R710 qui est le cerveau de tout d'Exvoid Network.
